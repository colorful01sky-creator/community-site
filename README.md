# 街づくりラボ ASOBI-X
　本サイトは、地域活動団体のウェブサイトを想定して制作した架空のプロジェクトです。
地域コミュニティの活性化を目的としたサイトの設計・実装スキルを証明するために制作しました。

## 🚀 公開URL
[https://colorful01sky-creator.github.io/portfolio-site/](https://colorful01sky-creator.github.io/community-site/)

- このサイトに登場する団体・イベント等はすべて架空のものです。
- 実在の団体とは一切関係ありません。

## 🛠 使用技術
- HTML5 / CSS3 (レスポンシブ対応)
- レスポンシブデザイン（PC・タブレット・スマホ対応）
- GitHub Pages（ホスティング）

## ✨ こだわったポイント
- **実務を想定した情報設計**: 架空のプロジェクトながら、実際の団体運営で必要となる「活動報告」「イベント案内」「参加導線」を想定したコンテンツ構成を行いました。
- **親しみやすいUIデザイン**: 地域の幅広い世代が利用することを想定し、温かみのある配色と、迷わないナビゲーション設計を意識しました。
- **保守性の高いコーディング**: 運用の継続を想定し、CSSのクラス命名や構造を整理して、後からの修正が容易なコードを記述しています。

## 📂 フォルダ構成
- `index.html`: トップページ（ビジョン・お知らせ）
- `activity.html`: 活動内容詳細（活動実績の紹介）
- `css/`: スタイルシートを格納
- `image/`: サイト内で使用している画像一式
